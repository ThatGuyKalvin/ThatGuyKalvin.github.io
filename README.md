# Portfolio

This is the source code for my personal portfolio webapp.

Dependencies:
- vue/cli: v5.0.8
- npm: v9.3.1
- node: v18.14.1

## Install
```
npm ci
npm run dev
```

## Lint
```
npm run lint
```

## Prettier
```
npm run format
```

## Build
```
npm run build
```

## Test
```
npm run test:unit
npm run test:e2e
npm run test:e2e:dev
```
